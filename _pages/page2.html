How it works

Passengers

  1. Select option - Passenger / Driver
  2. Select date of travel (can highlight multiple days if flexible)/li>
  3. Select destination – (vehicles will be highlighted in green on the map that are traveling to your destination / compare cost to make the journey)
  4. Pull up profile of the driver – Name / Age / Vehicle / Lifetime mission / Ratings
  5. Submit request – Pick-up point / Timings / Luggage / Smoking / Contact number / Other comments

Drivers

  1. Select option - Passenger / Driver
  2. Select date of travel (can highlight multiple days if flexible)
  3. Select destination / destinations if happy to stop off at multiple places en route
  4. Select pick – up point / specify if they are happy to collect within a 5 mile radius
  5. Set requirements – No of places available / luggage allowance / smoking (Y/N) / other comments
