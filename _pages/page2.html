How it works

Passengers

  • Select option - Passenger / Driver
  • Select date of travel (can highlight multiple days if flexible)/li>
  • Select destination – (vehicles will be highlighted in green on the map that are traveling to your destination / compare cost to make the journey)
  • Pull up profile of the driver – Name / Age / Vehicle / Lifetime mission / Ratings
  • Submit request – Pick-up point / Timings / Luggage / Smoking / Contact number / Other comments

Drivers

  • Select option - Passenger / Driver
  • Select date of travel (can highlight multiple days if flexible)
  • Select destination / destinations if happy to stop off at multiple places en route
  • Select pick – up point / specify if they are happy to collect within a 5 mile radius
  • Set requirements – No of places available / luggage allowance / smoking (Y/N) / other comments
